NodeJs is a required dependency if you plan to use a JavaScript frontend. In order to install it, you must go to [https://nodejs.org](https://nodejs.org) and click the **Download** button.

![Download Nodejs Installer](/introduction/prerequisites/windows/node.png)

Once the installer is done downloading, click on it to open the installation wizard. Go through all of the wizards installation steps and use all of the recommended options.
